# MVP Scope Summary

Release commit: `8795489`

## In Scope

- Local desktop app.
- Local SQLite database.
- Local app-managed file storage.
- Admin setup, users, master data, audit, backup, restore, packaging.
- Secretary document filing, attachments, scan intake, document status, visibility, trash, PDF export.
- Staff/Head Viewer public read-only browsing and export of visible public documents.
- Role-based route and command guards.

## Out Of Scope

- OCR.
- Direct scanner hardware integration.
- Cloud sync.
- Web server/API server deployment.
- Multi-machine networking.
- Email workflow.
- New filing workflows beyond approved MVP slices.

## Build Order Completed

- Slice 1: Foundation and Authentication Shell.
- Slice 2: Master Data Management.
- Slice 3: User Management and Profile.
- Slice 4: Document Filing and Viewer Browsing.
- Slice 5: Document Visibility and Trash.
- Slice 6: Document Move and Status Management.
- Slice 7: Scan Intake.
- Slice 8: Audit Log and Activity Viewer.
- Slice 9: PDF Export and Attachment Preview.
- Slice 10: Backup, Restore, and Installer Packaging.
