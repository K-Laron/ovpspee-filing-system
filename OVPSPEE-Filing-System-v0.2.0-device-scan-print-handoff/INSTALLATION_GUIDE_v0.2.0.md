# Installation Guide v0.2.0

This release is source-tagged as `v0.2.0-device-scan-print` at commit `b9e4616`.

## Install Path

Use the existing OVPSPEE Filing System installation process from the MVP release handoff unless a newer installer is separately built and approved.

No installer rebuild was performed for this handoff package.

## Device Requirements

- Windows environment for scanner/printer detection.
- WIA-compatible scanner for scan capture.
- Windows printer configured for document printing.
- Local PDF print handler or Edge/WebView print fallback available for document PDF printing.

## Post-Install Checks

- Open Admin Devices.
- Refresh scanner/printer detection.
- Save default scanner/printer if desired.
- Open Secretary Scan Intake and confirm scanner panel appears.
- Open a public document as Staff/Head Viewer and confirm print/export controls appear.
